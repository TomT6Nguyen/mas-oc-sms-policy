MAS-OC Alerts — SMS Opt-In & Consent

Last updated: 2026-06-28

MAS-OC Alerts is an automated, application-to-person (A2P) notification service. The MAS-OC application sends one-way, system-generated operational alerts to the owner-operator who has enrolled inside the application. It is not an interactive or person-to-person (P2P) messaging service — the application is the sender, the owner-operator is the recipient, and recipients cannot use it to message anyone. Replies are limited to STOP/HELP handling.

What messages are sent: automated operational alerts triggered by events the application monitors in the owner-operator's own accounts — e.g. an inbound email matching a watchlist, an upcoming calendar event, or a monitored Google Sheet value changing. Transactional/operational only; never marketing.

How consent is captured (in-application opt-in)

There is no public sign-up form and no third-party recipients. The owner-operator opts in inside the authenticated MAS-OC application:

  1. The operator signs in to the password-protected MAS-OC configuration screen.
  2. They enter their own mobile number and verify it via a one-time code.
  3. They read the SMS consent disclosure and explicitly enable the "Operational SMS Alerts" channel (the checkbox shown below).
  4. The consent action and timestamp are recorded in the application.
  5. Upon enabling, MAS-OC sends a one-time confirmation SMS to the verified number ("You're now subscribed to MAS-OC operational alerts… Reply HELP for help, STOP to unsubscribe").

The exact consent screen the operator sees and affirmatively accepts:

MAS-OC  ›  Settings  ›  Operational SMS Alerts
Mobile number (yours)
✓ Verified via one-time code

Message frequency & rates

Message frequency varies based on system activity; messages are recurring. Message and data rates may apply.

Opt-out & help

Reply STOP at any time to unsubscribe — the operator receives a confirmation and no further messages. Reply HELP for help. The channel can also be disabled in the same configuration screen.
