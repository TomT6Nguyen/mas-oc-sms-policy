# mas-oc-sms-policy

Public privacy policy + terms for the **MAS-OC Alerts** SMS program, hosted on GitHub Pages,
for Twilio A2P 10DLC / toll-free campaign registration.

- `privacy.html` — Privacy Policy (non-sharing of mobile numbers, message frequency, "message and data rates may apply", STOP/HELP)
- `terms.html` — Terms & Conditions
- `index.html` — landing page linking both
- `.nojekyll` — serve files as-is (no Jekyll processing)

**No secrets. Safe to be public.** This repo is intentionally separate from any private code.

Before relying on it: replace `CONTACT_EMAIL_HERE` in `privacy.html` + `terms.html` with a real contact address.
